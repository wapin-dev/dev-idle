# Icônes DevIdle Agency (100 % local, gratuit)

Les icônes sont des SVG **locaux** pour rester **offline-friendly** et **gratuits** (aucune API payante).

- **Remplacer par Game-icons** : tu peux télécharger des SVG depuis [game-icons.net](https://game-icons.net) ou le [repo GitHub](https://github.com/game-icons/icons), les renommer selon les noms utilisés ici (user, users, coins, document, error, refresh, home, cart, list, flash, logout, settings, student, developer, senior, briefcase, contract, star, placeholder) et les déposer dans ce dossier.
- **Couleur** : les SVG fournis utilisent `stroke="#C4B5FD"` pour être lisibles sur le thème sombre. Tu peux changer en `currentColor` si tu préfères hériter de la couleur du texte.

Noms de fichiers attendus : `user.svg`, `users.svg`, `coins.svg`, `document.svg`, `error.svg`, `refresh.svg`, `home.svg`, `cart.svg`, `list.svg`, `flash.svg`, `logout.svg`, `settings.svg`, `student.svg`, `developer.svg`, `senior.svg`, `briefcase.svg`, `contract.svg`, `star.svg`, `placeholder.svg`.
